php如何安装redis扩展

博客
  • php
  • redis

概要

项目上用到redis,之前使用的memcache，二者其实差不多。但是java同事偏向于redis.于是我们也需要使用了。

redis的优势

  1. Redis不仅仅支持简单的k/v类型的数据，同时还提供list，set，zset，hash等数据结构的存储。

  2. Redis支持数据的备份，即master-slave模式的数据备份。

  3. Redis支持数据的持久化，可以将内存中的数据保持在磁盘中，重启的时候可以再次加载进行使用。

redis的参考文档

http://www.runoob.com/redis/redis-php.html

linux下安装

$ wget https://github.com/phpredis/phpredis/archive/2.2.4.tar.gz
$ cd phpredis-2.2.7                      # 进入 phpredis 目录
$ /usr/local/php/bin/phpize              # php安装后的路径
$ ./configure --with-php-config=/usr/local/php/bin/php-config
$ make && make install

还需要修改php.in文件

vi /usr/local/php/lib/php.ini
extension=redis.so

心得体会

  1. 掌握了一个新的东西。phpize是用来扩展php扩展模块的，通过phpize可以建立php的外挂模块。
  2. ./configure –with-php-config 是用来编译之前的准备工作，控制编译的结果。configure是一个shell脚本，它可以自动设定源程序以符合各种不同平台上Unix系统的特性，并且根据系统叁数及环境产生合适的Makefile文件或是C的头文件(header file)，让源程序可以很方便地在这些不同的平台上被编译连接。
  3. make 和make install 表示编译，然后安装。
  4. 这里只是安装了扩展，而非redis服务。redis服务需要单独处理。

测试是否连接

//连接本地的 Redis 服务
  $redis = new Redis();
  $redis->connect('127.0.0.1', 6379);
  echo "Connection to server sucessfully";
        //查看服务是否运行
  echo "Server is running: " . $redis->ping();

读取与设置

//连接本地的 Redis 服务
   $redis = new Redis();
   $redis->connect('127.0.0.1', 6379);
   echo "Connection to server sucessfully";
   //存储数据到列表中
   $redis->lpush("tutorial-list", "Redis");
   $redis->lpush("tutorial-list", "Mongodb");
   $redis->lpush("tutorial-list", "Mysql");
   // 获取存储的数据并输出
   $arList = $redis->lrange("tutorial-list", 0 ,5);
   echo "Stored string in redis";
   print_r($arList);

其他

到此php的Linux下redis扩展安装ok。 window下扩展更容易。只需要去对应的网址下载匹配的php版本的dll即可。

http://pecl.php.net/package/redis/2.2.7/windows

下载好的dll，存放到php的ext目录，修改php.ini，添加 extension=php_redis.dll
